# Timesheet Bot

## Project Documentation

The Timesheet Bot is a tool designed to help users track time spent on various tasks.

### Features
- Automatic time tracking
- Task management
- Reporting capabilities

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/sithortodox/timesheet_bot.git
   cd timesheet_bot
   ```
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure the environment variables:
   ```bash
   cp .env.example .env
   ```
   Edit the `.env` file to set your configurations.
4. Run the bot:
   ```bash
   python bot.py
   ```
